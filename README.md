# GitHub User Finder

A simple React application to search and view GitHub user profiles using the GitHub API.

## Features
- Search GitHub users by username
- View profile picture, name, bio, followers, and repositories
- Error message for invalid usernames
- Dark theme UI

## Technologies Used
- React
- JavaScript
- GitHub API
- HTML
- CSS

## How to Run
1. Install dependencies  
   `npm install`
2. Start the app  
   `npm run dev`
